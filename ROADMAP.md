# Note All 研发路线图 (Roadmap)

基于竞品分析与核心理念“无感收集、AI 即刻懂”，制定以下新功能规划及研发路线图。

## 📅 近期开发计划

### Phase 1: 浏览器剪藏扩展 (Browser Extension)
* **状态**: ✅ 已完成
* **成果**: 补全了 PC 工作流中的网页信息抓取拼图。支持划词右键剪藏及 Popup 快捷输入。

### Phase 2: VLM (视觉大模型) 图像深度语义感知
* **状态**: ✅ 已完成
* **成果**: 接入多模态大模型，实现了图片内容的深度描述与 OCR 文本的智能融合。

### Phase 3: 智能记忆拼图 (Serendipity Review)
* **状态**: ✅ 已完成
* **成果**: 实现了“灵感碰撞”功能，AI 自动串联随机碎片生成启发性随笔，盘活了死知识。

### Phase 4: 自动隐式双链 (Auto-Linking)
* **状态**: ✅ 已完成
* **成果**: 建立了基于标签交集的关联发现机制，碎片详情页可自动提示相关内容，实现隐式双链。

---
*注：遵循渐进式开发原则，我们将依照上述 Phase 的顺序依次展开。*
