# 用户使用手册

本指南将帮助你充分利用 Note All 强大的收集能力和 AI 功能。

## 📥 极速收集 (Collection)

### Android 手机端
- **系统分享**：在任何浏览器、相册或 App 中点击“分享”，选择 Note All 即可秒级收录。
- **剪贴板监听**：打开 App 时，它会自动识别剪贴板中的内容并提示你是否一键保存。

### Windows 电脑端
- `Alt+Q`：截取屏幕任意区域。Note All 会自动进行 OCR 识别，并根据文字内容进行 AI 摘要和标签提取。
- `Alt+Shift+Q`：弹出迷你闪记窗口，随时记录灵光一现。

### 浏览器 (Chrome/Edge)
- 使用浏览器插件剪藏网页全文，或选中划词进行精准收集。

## 🧠 AI 自动化处理 (Processing)

你存入的每一条笔记都会经过以下自动化流程：
1. **OCR 流水线**：从图片中提取文字，完美支持多语言、代码块和数学公式。
2. **LLM 智能摘要**：自动为笔记生成核心摘要，让你在不阅读全文的情况下掌握重点。
3. **自动打标**：AI 会根据内容深度提取主题特性，实现无感分类。

## 🧪 知识实验室 (Knowledge Lab)

知识实验室旨在实现“从碎片的收集到知识的升华”。
1. 在主列表中勾选多条相关的笔记或片段。
2. 点击 **“合成” (Synthesize)**。
3. AI 将自动分析素材间的因果、逻辑和矛盾点，生成一篇结构化的 Markdown 长文。
4. 确认后可将其保存为一篇“合成笔记”，并保留原始碎片的溯源链接。

## 🤖 微信助手 (WeChat Bot)

你可以通过微信直接与你的个人知识库对话：
1. 在 Web 控制台的 **WeChat** 标签页扫码登录，完成账号绑定。
2. 直接向机器人发送文字、链接或图片进行收录。
3. 随时提问，例如：“我上周存的那篇关于 Docker 的文章讲了什么？”机器人将基于你的笔记库进行精准回答。

## 🔍 语义搜索与图谱

传统搜索只看关键词，而 Note All 的搜索理解**意图**。
- 尝试搜索“关于提高生产力的想法”，即使笔记里没有“生产力”三个字，系统也能通过含义找到相关内容。
- 使用 **“知识图谱” (Knowledge Graph)** 视觉化查阅笔记间的联系。
  - **显式双链**：在编辑时使用 `[[笔记标题]]` 语法手动建立关联。
  - **幽灵节点**：如果引用的笔记尚未创建，图谱中会显示为“幽灵节点”，点击可快速创建该笔记。
  - **隐式关联**：系统会自动基于标签和语义相似度在图谱中连线。
